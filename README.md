# NovelBin Web Scraper

A simple Python scraper for downloading web novel chapters from NovelBin and converting them into EPUB.

## Features
- Fetch full chapter list
- Download custom chapter ranges
- Save chapters as numbered `.txt` files
- Log failed chapters
- Build EPUB with chapter TOC

## Install
pip install requests beautifulsoup4 lxml ebooklib

## Use
- run main.py and give the novelbin url
- run epub_create to get an epub



## Warning
Don't go crazy with it -_-, too many requests could cause issues.
